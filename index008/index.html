CSS Селектор Пример Описание
.class .myClass Выбирает все элементы с классом myClass(class="myClass")
#id #main Выбираем элемент с идентификатором main(id="main")
* * Выбирает все элементы
элемент div Выбор всех элементов <div>
элемент, элемент div, span Выбор всех элементов <div> и всех элементов <span>
[атрибут] [href] Все элементы с атрибутом href
[атрибут="значение" [href="https://vk.com/"] Выбирает все элементы с атрибутом href, значение которого ТОЧНО совпадает со значением, указанным селектором
[атрибут*="значение"] [href*="vk.com"] Выбирает все элементы с атрибутом href, в значениии которого(в любом месте) встречается подстрока(в виде отдельного слова или его части) "vk.com"
элемент элемент .div .span Выбор всех элементов <span> внутри <div>
элемент+элемент div+p Выбирает все элементы <p>, которые расположены сразу после элементов <div>
элемент~элемент p~div Выбирает все элементы <div>, которые расположены сразу перед элементом <p>
элемент: nth-child(номер) div:nth-child(3) Выбирает третий элемент, который лежит внутри <div>
